Simple Python Calculator

Quick Start
- Run interactive mode: `python calculator.py`
- Run via CLI: `python calculator.py <operation> <a> <b>`

Operations
- `add`: addition
- `sub`: subtraction
- `mul`: multiplication
- `div`: division (raises error on divide-by-zero)

Examples
- `python calculator.py add 2 3` → 5
- `python calculator.py sub 10 4` → 6
- `python calculator.py mul 7 8` → 56
- `python calculator.py div 9 2` → 4.5

Testing
- From the `python-calculator` folder, run: `python -m unittest -q`
